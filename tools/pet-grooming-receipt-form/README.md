# 寵美現場服務簽收單

A4 橫式手寫簽收單兩款，店內員工現場填寫 + 客戶簽收，老闆娘收班拍照傳給 Cowork OCR 識別後 KEY IN 進 Google Sheets。

## 📋 兩張單分別給誰用

| 簽收單 | 適用對象 | KEY IN 目標 |
|---|---|---|
| `散客服務簽收單.xlsx` | 未預約臨時散客 + 一般預約客（現金/匯款/LINE Pay 結帳） | 檔案 A「服務紀錄」分頁 |
| `儲值客戶服務簽收單.xlsx` | 已儲值客戶（餘額扣抵服務 / 現場新增儲值） | 檔案 B「交易明細」分頁 |

> **判斷哪張：** 客人有沒有儲值？有 → 儲值單；沒有 → 散客單。一律以「結帳方式」為準，不以「有沒有預約」分。

## 🖨️ 列印

兩個 .xlsx 都已內建設定：A4 橫式、邊界已調、列印區域已設、不用縮放。

1. 打開 .xlsx
2. Ctrl+P
3. 印 A4 橫式即可

每張單可填 15 列。一天散客約多少就印多少張、儲值客同理。預先印 10-20 張放在現場備用。

## ✍️ 員工填寫 SOP

**頁首每天先寫**（一張單只寫一次，不要每列重寫）：
- 日期：今天的年月日
- 美容師：今天值班的人

**每筆服務一列**，由左至右填：

### 散客單
| 欄位 | 怎麼填 |
|---|---|
| # | 預印 1-12，不用填 |
| 主人電話 | 10 碼，開頭 0 不能省（例：0905-435-751） |
| 寵物名 | 短名即可（例：豆豆） |
| 服務項目 | 在 ☐ 上打勾 ✓ 或塗滿，可複選；「其他」勾起來再寫項目 |
| 金額 | 純數字（800、1500），不寫 NT$、不寫「元」 |
| 付款方式 | 在 ☐ 上打勾 ✓ |
| 簽收 | 客戶現場簽名 |
| 備註 | 有特殊狀況才寫（例：免清耳、毛球嚴重） |

### 儲值客戶單
| 欄位 | 怎麼填 |
|---|---|
| # | 預印 1-12，不用填 |
| 主人電話 | 10 碼，開頭 0 不能省 |
| 客戶姓名 | 短名即可 |
| 服務項目 | 同散客單，勾選 |
| 消費金額 | 純數字（這次扣抵多少） |
| 儲值金額 | **大多空白**！只在客戶現場新增儲值才填（例 5000 / 8000） |
| 扣後餘額 | 扣抵後剩多少（例：原 3200，扣 800 → 寫 2400） |
| 簽收 | 客戶現場簽名 |
| 備註 | 有特殊狀況才寫 |

> 💡 **餘額怎麼查？** 員工查詢頁輸入電話 → 看儲值金餘額欄位。

## 📸 老闆娘收班流程

1. 確認當天所有列都填完、客戶都簽收完
2. 兩張單分開拍清楚（不要折角、不要反光）
   - 散客單拍一張
   - 儲值單拍一張
3. 用 LINE / Drive 傳給 Cowork
4. 順手把今天那張紙本歸檔到月份夾，紙本是「客戶確實簽收」的證據要留 1-2 年

## 🚨 容易踩坑

- **電話開頭 0 千萬不能省。** Sheets 那邊 4/28 才剛踩過 09xxx 被自動轉成數字吃掉前 0 的坑（已修），但手寫時若就漏了 0，KEY IN 進去也救不回。
- **金額只寫純數字。** NT$、元、逗號（1,500）都不要寫，KEY IN 時要丟到 Sheets 數字欄。
- **勾選用 ✓ 或塗滿。** 不要在 ☐ 上畫斜線（OCR 容易判錯）。
- **服務項目要勾的不要省。** 多項目就多勾，免得月底對不出來。
- **儲值客戶單的「儲值金額」欄大多空白。** 只在新增儲值時才填，純消費扣抵不要填。

## 🛠️ 改格子

服務項目要加「染色」「剃毛」？付款方式要加「街口支付」？要每張印 20 列不要 15 列？打開 `generate_forms.py`，改最上面常數區：

```python
SHOP_NAME = "洗毛這件小事"
SERVICE_OPTIONS = ["洗澡", "全美容", "剪指甲", "SPA"]
PAYMENT_OPTIONS = ["現金", "匯款", "LINE Pay"]
PRIMARY_ROW_COUNT = 15      # 主版本列數（一頁印幾列）
EXTRA_ROW_COUNTS = []       # 想多產一個備援列數版本就加進來，例如 [20]
```

改完重跑：
```bash
python generate_forms.py
```

兩個 .xlsx 會被覆蓋成新版。重印即可。
